What's this?

As stated in the home page, DF: Lost Memories is a website where I (High Flyer) document unused content I find in DONTFORGET, all the way from the 2018 demo to the latest version of the game (the Old Beta and Classic Multiplayer not included).

The videos and screenshots here will mostly be by me. The authors will be credited otherwise.

All articles are categorized based on which version the content originated from.

I also use a different vessel and color for each category, to make them easier to recognize at a glance. I use Frisk (Dark Purple) for Demo 2, Harlow (Purple) for Chapter 1, Kris (Blue??) for Old DFC, and Quote (Orange) for New DFC.

Content exclusive to Demo 2 is listed on a separate page, known as Lost Timelines.

How can I help?

By finding unused content? You can also check to see if any of the unused content in available in versions older than their listed "first version" or newer than their "last version".

You can also make room ID lists for versions that don't have them. It might not be incredibly useful for v2.X versions but one for v1.3.3 would be much appreciated.

Finally, if you have any old DF versions archived, please send them to me. Especially...

  • Android ones, as there's already an archive of almost every Windows v2.X build.
  • Older Chapter 1 builds. Specifically, v1.0.0 would be very nice to have, so I can know if some things have been in the game since the very beginning or not.
  • Older Demo 2 builds, for the same reason as above. v0.1.0 would be really cool.

If you're on this website, you might already know me on Discord. My username is, unsurprisingly, hflyer.

Well how DO I find unused content

That's honestly a good question. If I were you I would just go back to older versions and edit the save file to go to different room IDs. I haven't really explored a lot of versions other than v1.3.3, v2.4.3b and v2.7.9c.

Or, if you know what you're doing, you might be able to take a peek at the game's files. However...

There are rules

  • No content ripped straight from the files. If something is not accessible in-game through "legitimate" methods like save editing, noclipping, the map editor, or any other way that doesn't involve straight up modding the game, then it won't be posted here.
  • No decompiling. As of v2.7.10b this is impossible anyway, but the "files" I keep mentioning don't give you access to the code in earlier versions. As such, unused content that's found by looking at decompiled code will not be accepted.

Note: For Demo 2, sharing content from the files (and modding, to an extent) for documentation's sake is allowed.
Mods for this version may not be distributed, however.

That's about it. Have fun!
