In the files of Demo 2, there's an unused version of the UI used by the machine.

First, a list of things that were modified in the code to make the video possible:

  • The machine was changed to create obj_newmap instead of obj_selecttimeline.
  • obj_soulpieces was changed to work with obj_newmap instead of obj_selecttimeline, and not automatically adjust its position.
  • obj_instructions was integrated into the machine (trust me when I say it barely works).

It's worth noting that unlike in the final game, where they're referred to as timelines, here the different worlds are called alternate universes.

Everything after the second universe is locked. However, the third one's thumbnail can be seen showing the Beta Void while hovering over the fourth universe.

The help guide mentions Frisk's phone being upgraded. This relates to the Demo 2 Phone UI article.

I should also mention that there's some code suggesting that said help guide would've originally appeared after you travel to a different universe for the first time.

The second one

There's... another unused machine UI. This one is only a universe selection screen, like the final version, and includes details about some planned timelines.

For this one, the machine was modified to create obj_map_main, and obj_map_travel had its sprite changed to spr_map_travel.

It's worth noting that this version can go to the Beta Void without any modifications.

The six empty universe slots can also be full.

Four of them are been marked with "g" (for Gaster?) in the code and would've been unlocked when their variables were set (they aren't set by anything in the final version).

The other two use the letter "m", for multiplayer, and would've been accessible in online mode. These two are the Classic Void, and what probably would've been this game's Gold Grind.
